# TalkMate 第一阶段任务拆解（MVP）

> 项目名称：TalkMate  
> 阶段：Phase 1 / MVP  
> 目标：完成从登录到对话到课后总结的端到端闭环  
> 日期：2026-06-04  

---

## 阶段目标

MVP 阶段完成后，用户能完成以下完整流程：
**登录 → 选择场景 → 语音对话（≥10 轮）→ 查看纠错 → 查看课后总结 → 查看历史练习记录**

## 任务详情

### T-001：项目结构搭建 + 用户注册登录

**任务目标**：搭建前后端项目骨架，实现用户注册、登录、登出功能。

**范围边界**：
- 本次包含：项目目录结构、前端路由、后端 API 服务、注册/登录接口、JWT 会话、登录页/注册页
- 本次不包含：第三方登录、密码找回、用户资料编辑

**目录 / 模块**：
- 允许修改目录：`frontend/`（auth 相关）、`backend/`（auth 模块）、`docs/`（API 文档）
- 目标模块：auth
- 是否新增文档：是（`docs/api/auth.md`）

**验收标准**：
- [ ] 用户能用账号密码注册
- [ ] 用户能用账号密码登录
- [ ] 登录后刷新页面保持登录状态
- [ ] 登出后清除会话

**测试重点**：注册重复用户名处理、错误密码提示、JWT 过期

**依赖与风险**：密码必须 bcrypt 加密，不可明文存储

---

### T-002：场景选择页面

**任务目标**：实现首页场景卡片列表，用户能选择场景进入对话。

**范围边界**：
- 本次包含：场景数据（3-5 个预设场景）、场景卡片 UI、场景详情接口
- 本次不包含：自定义场景、场景编辑

**目录 / 模块**：
- 允许修改目录：`frontend/src/pages/scenarios/`、`backend/app/modules/scenario/`
- 目标模块：scenario

**验收标准**：
- [ ] 首页展示 3-5 个场景卡片
- [ ] 点击场景卡片能进入对话页面
- [ ] 场景数据通过后端 API 获取

**测试重点**：场景卡片布局、空态展示

**依赖与风险**：无

---

### T-003：场景对话页面（文字模式）

**任务目标**：实现场景对话页面，支持文字输入对话。

**范围边界**：
- 本次包含：对话消息列表、文字输入框、消息发送、对话创建接口
- 本次不包含：语音输入、AI 智能回复（先用固定回复占位）

**目录 / 模块**：
- 允许修改目录：`frontend/src/pages/conversation/`、`backend/app/modules/conversation/`
- 目标模块：conversation

**验收标准**：
- [ ] 能从场景页进入对话页面
- [ ] 能输入文字并发送消息
- [ ] 消息列表正确展示对话记录
- [ ] 刷新后对话记录不丢失

**测试重点**：消息发送/接收状态、长文本截断、滚动行为

**依赖与风险**：AI 集成未完成前用固定回复占位

---

### T-004：AI 对话集成

**任务目标**：接入 DeepSeek API，实现场景化 AI 回复。

**范围边界**：
- 本次包含：DeepSeek API 调用封装、prompt 模板（按场景）、消息发送后调用 AI
- 本次不包含：流式回复、多轮上下文优化

**目录 / 模块**：
- 允许修改目录：`backend/app/modules/ai_service/`
- 目标模块：ai-service
- 是否新增文档：是（`docs/architecture/ai-prompt-design.md`）

**验收标准**：
- [ ] 发送消息后 AI 能在 2 秒内回复
- [ ] AI 回复与当前场景相关
- [ ] AI 回复语言自然流畅

**测试重点**：不同场景 prompt 效果、API 超时处理、错误降级

**依赖与风险**：DeepSeek API key 已通过环境变量配置

---

### T-005：STT/TTS 语音交互

**任务目标**：接入浏览器 Web Speech API，实现语音输入和语音播放。

**范围边界**：
- 本次包含：语音识别（SpeechRecognition）、语音合成（SpeechSynthesis）、录音按钮 UI
- 本次不包含：流式语音、外部 STT/TTS API

**目录 / 模块**：
- 允许修改目录：`frontend/src/components/`（语音组件）
- 目标模块：stt-tts

**验收标准**：
- [ ] 点击录音按钮能识别语音并转为文字
- [ ] AI 回复文字能自动播放语音
- [ ] 说话后 1 秒内显示文字

**测试重点**：不同浏览器兼容性（Chrome/Safari）、录音权限提示

**依赖与风险**：浏览器 API 兼容性，需测试 Chrome 和 Safari

---

### T-006：纠错反馈 + 课后总结

**任务目标**：对话结束后生成纠错反馈和课后总结。

**范围边界**：
- 本次包含：AI 生成纠错反馈、AI 生成课后总结、总结展示页面
- 本次不包含：历史对比、进步曲线

**目录 / 模块**：
- 允许修改目录：`frontend/src/pages/summary/`、`backend/app/modules/conversation/`
- 目标模块：feedback, conversation

**验收标准**：
- [ ] 对话结束后展示纠错反馈（至少 1 条）
- [ ] 展示课后总结（评分、优点、改进建议）
- [ ] 总结数据持久化到数据库

**测试重点**：总结生成逻辑、总结页面布局

**依赖与风险**：依赖 T-004 AI 集成

---

### T-007：练习记录列表 + 对话历史

**任务目标**：首页展示历史练习记录，支持回溯查看对话详情。

**范围边界**：
- 本次包含：练习记录列表（场景名、日期、评分）、点击记录查看对话详情
- 本次不包含：筛选/排序、数据导出

**目录 / 模块**：
- 允许修改目录：`frontend/src/pages/home/`、`backend/app/modules/conversation/`
- 目标模块：profile

**验收标准**：
- [ ] 首页能查看历史练习记录列表
- [ ] 点击记录能回溯查看完整对话
- [ ] 列表展示场景名、日期、评分

**测试重点**：空态展示、分页/加载更多

**依赖与风险**：无

---

### T-008：响应式 UI 适配

**任务目标**：所有页面适配移动端和 PC 端，一套 UI 覆盖 375px-1440px。

**范围边界**：
- 本次包含：所有已有页面的响应式布局调整
- 本次不包含：深色模式、主题切换

**目录 / 模块**：
- 允许修改目录：`frontend/src/`（样式相关）
- 目标模块：所有前端模块
- 是否新增文档：否

**验收标准**：
- [ ] 375px（手机）、768px（平板）、1440px（PC）均可正常使用
- [ ] 核心交互在手机端可单手操作
- [ ] 无水平滚动条、无文字溢出

**测试重点**：多视口测试、触摸交互

**依赖与风险**：无
