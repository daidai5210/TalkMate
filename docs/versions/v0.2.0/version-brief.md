# TalkMate v0.2.0 版本概要

> 路线：B — 旧项目功能迭代
> 当前运行版本：v0.1.0（主线 commit `898bfbd`）
> 目标版本：v0.2.0

## 版本目标

新增"中文母语错误画像 + 个性化复练闭环"功能。让用户知道自己"中式英语"的具体问题和类型分布，基于画像推荐下一次练习目标，形成"发现弱点→定向练习→再检测"的闭环。

## 版本范围

| 维度 | 内容 |
|---|---|
| 核心功能 | 中文母语错误画像（5 类）+ 个性化复练推荐 |
| 入口 | 总结页 + 首页轻量入口 |
| 错误类型 | 中式语序、时态、冠词、介词、直译表达（固定 5 类） |
| 推荐策略 | 每次只推荐 1 个下一次练习目标 |
| 画像聚合 | 按最近 5 次练习聚合 |
| 不做项 | 独立能力画像页、多语言、课程系统、社交功能、复杂数据脚本 |
