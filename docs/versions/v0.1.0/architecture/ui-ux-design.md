## 三、页面布局与交互

### 1. 登录页 `/login`

**桌面（1440px）：** 居中卡片，Logo → 用户名 → 密码 → 登录按钮 → "去注册" 链接  
**移动端（375px）：** Logo 缩小，表单全宽，按钮 sticky bottom

**交互状态：**
- Loading：按钮 Spinner，表单禁用
- Error：输入框变红，下方错误文案
- 登录成功自动跳转首页

### 2. 注册页 `/register`

**布局：** Logo → 用户名 → 密码（含强度条）→ 确认密码 → 验证码（4 位数字）→ 注册按钮 → "去登录" 链接

**交互状态：**
- 密码强度实时指示（弱/中/强）
- 确认密码不一致时标红
- 注册成功自动跳转登录页

### 3. 首页 `/`

**桌面（1440px）：** 顶部导航栏（Logo + 用户头像/退出）→ "选择你的练习场景" 标题 → 5 个场景卡片网格（2-3 列）→ "我的练习记录" 列表

**移动端（375px）：** 导航栏简化，场景卡片竖向排列，练习记录列表

**每场景卡片：** 图标 + 场景名 + 描述 + "开始练习" 按钮  
**每条记录：** 场景名 + 日期 + 评分 + "查看详情"

**交互状态：**
- Loading：场景骨架屏
- Empty：无记录时插画 + "开始你的第一次练习吧"
- Error：加载失败提示 + "重试"

### 4. 对话页 `/conversation/:id`

**桌面（1440px）：** 顶部场景名 + "结束对话" → 中间消息列表（可滚动）→ 底部输入区（语音按钮 + 文字输入框 + 发送）

**消息气泡：** 用户消息右侧浅蓝色，AI 消息左侧浅灰色，附时间戳

**语音交互流程：**
1. 点击麦克风 → 开始录音 → 按钮变红 + 波形动画
2. 停止说话 → 自动转文字 → 显示在输入框
3. 发送 → 用户消息气泡出现 → Loading "正在思考..."
4. AI 回复 → 气泡出现 → 自动播放语音（TTS）

**交互状态：**
- Loading：首次骨架屏，AI 回复中 "正在思考..."
- Empty：新对话场景描述 + "开始对话吧"
- Error：AI 失败提示 + "重试"

### 5. 总结页 `/conversation/:id/summary`

**桌面（1440px）：** 顶部场景名 + "返回主页" → 综合评分卡片（大数字 + 环形进度条）→ 纠错反馈列表（展开式：原文→正确→解释→改进）→ 词汇分析卡片 → 改进建议卡片 → "再次练习" 按钮

**交互状态：**
- Loading：生成中骨架屏 + "正在分析..."
- Empty：无纠错时 "太棒了，没有发现错误！"
- Error：生成失败 + "重新生成"

## 四、主题支持

- 默认浅色主题，支持深色主题切换（localStorage 持久化）
- 所有颜色使用 design tokens，禁止硬编码色值
