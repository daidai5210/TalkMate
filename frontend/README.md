# TalkMate Frontend (React + Vite + TS)

## 启动

```bash
cd frontend
npm install
cp .env.example .env
npm run dev
```

访问 <http://localhost:5173>

## 技术栈

- React 18 + TypeScript 5
- Vite 5
- Tailwind CSS 3
